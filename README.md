# 🏥 NivaranAI

### *Intelligent Pre-Consultation & Voice-to-SOAP Healthcare System*

**“Killing the Waiting Room Lottery”**

## 🚀 Overview

NivaranAI is an AI-powered healthcare assistant that transforms waiting rooms into intelligent triage systems. It interacts with patients using voice in their local language, understands symptoms, and generates structured medical notes for doctors—before consultation even begins.

## 💡 Problem

* Doctors spend **70% time on documentation**
* Language gap between patients and doctors
* No prioritization → critical patients wait longer

## ✅ Solution

NivaranAI automates pre-consultation with:

* 🎤 Voice-based patient interaction
* 📄 Auto-generated SOAP notes
* 🚨 Smart triage system (priority-based queue)
* 📲 WhatsApp prescription delivery

## ⚙️ How It Works

1. **Voice Kiosk**
   Patient speaks symptoms in local language

2. **AI Processing**
   AI asks follow-up questions & analyzes condition

3. **Doctor Dashboard**
   SOAP note + priority shown instantly

4. **Prescription**
   Doctor speaks → Patient receives via WhatsApp
